## Component Spacing & Sizing

### Navbar

```
Height: 80px (fixed)
Padding: 16px 24px
Content: Logo (left) | User Info (right)
Logo:
  - Main: 18px bold
  - Tagline: 12px regular
User Info:
  - Name: 14px
  - Button: 8px 16px (sm rounded, blue bg)
```

### Summary Cards

```
Grid: 3 columns (desktop), 1 column (mobile)
Gap: 16px
Card Size: ~280px-400px (responsive)
Padding: 24px
Border: Left 4px colored
Shadow: 0 4px 6px rgba(0,0,0,0.1)
Hover: Lift 4px, shadow enhance
```

### Study Plan Card

```
Padding: 24px
Border-radius: 12px
Sections:
  - Header: 2 columns (info | meta)
  - Progress: Label + bar
  - Actions: 2 buttons side-by-side
Gap Between: 16px
Mobile: Header stacks vertically
        Buttons stack vertically
```

### Roadmap Items

```
Height: 80px (auto)
Padding: 12px
Gap: 12px
Checkbox: 32x32px (circle)
Content: Left-aligned
Mobile: Reduced padding (12px -> 8px)
        Checkbox: 28x28px
```

### Primary Blue (#6366f1)

- Active tab background
- Primary buttons (Continue Learning)
- Mentor card accent border
- Navigation highlights
- Links and interactive elements

### Secondary Pink (#ec4899)

- Gradient backgrounds
- Secondary accents
- Hover effects

### Success Green (#10b981)

- Completed checkmarks in roadmap
- Easy difficulty badges
- Positive indicators

### Warning Amber (#f59e0b)

- Catch-up card accent
- Days countdown
- Medium difficulty badges
- Attention-needed items

- Text: #1f2937 (dark)
- Secondary text: #6b7280 (medium)
- Tertiary text: #9ca3af (light)
- Borders: #e5e7eb
- Background: #f9fafb

## Interactive States

### Buttons

```
Normal:   Color + shadow
Hover:   Darker color + enhanced shadow + lift
Active:  Color shift + pressed effect (translateY 0)
Focus:   Outline (accessibility)
```

### Tabs

```
Inactive: Gray text, transparent bg
Hover:   Light blue background
Active:  Blue background + text + bottom indicator
```

### Cards

```
Normal:  White bg + soft shadow
Hover:   Enhanced shadow + lift animation
Focus:   (For keyboard navigation)
```

## Mobile Touch Optimization

```
Button Height: Minimum 44px (accessibility)
Tap Target:    Minimum 48px
Padding:       Extra spacing around interactive elements
Text Size:     Minimum 14px (readable on mobile)
Spacing:       Increased gaps between elements
Overflow:      Horizontal scroll for overflowing content
```
